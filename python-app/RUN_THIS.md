## How to Run (Simple Steps)

### 1. Pull the Latest Code

```bash
git pull origin main
```

### 2. Activate Virtual Environment

```bash
venv\Scripts\activate
```

### 3. Run the Binance Version

```bash
python app_binance.py
```

### 4. Open Browser

Go to: **http://localhost:5000**

### 5. Add Trading Pairs

Try these popular pairs:
- **BTCUSDT** - Bitcoin
- **ETHUSDT** - Ethereum
- **BNBUSDT** - Binance Coin
- **SOLUSDT** - Solana
- **ADAUSDT** - Cardano

**They will add in 2-3 seconds with REAL prices!**
