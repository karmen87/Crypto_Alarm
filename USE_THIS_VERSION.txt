HOW TO USE:

2. Activate virtual environment:
   Windows: venv\Scripts\activate
   Mac/Linux: source venv/bin/activate

3. Run the FIXED version:
   python app_fixed.py

4. Open browser:
   http://localhost:5000

5. Add a pair (e.g., BTCUSDT):
   - You'll see "Fetching data... Please wait 5-10 seconds"
   - The connection will stay active
   - After 5-10 seconds, the pair will appear

6. Wait 10 seconds before adding another pair!

---

WHY IT TAKES 5-10 SECONDS:

CoinGecko's free API requires:
- 2 second delay between each API call
- Adding a pair makes 2-3 API calls
- Total time: 4-10 seconds

This is normal and expected with the free API!
